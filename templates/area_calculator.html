Calculate accurate areas for closed polygons and 3D regions

Area Calculation Tool (Prototype)

This specialized tool will provide accurate area calculations for all closed polygons and 3D regions based on their object classification. Calculate areas for buildings, sidewalks, parking lots, BMPs, accessible features, and other site elements.

Planned Features

  • Multi-select area calculation by object type
  • Support for 2D polygons and 3D closed regions
  • Area grouping by category (accessible features, BMPs, grading, etc.)
  • Unit conversion (SF, acres, hectares)
  • Export area takeoff reports
  • Integration with Material Volume Calculator and Pervious/Impervious Analysis

Under Construction

This tool is currently in development. Check back soon for the full implementation.

Supported Object Types

Accessible Features

BMPs
