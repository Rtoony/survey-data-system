About This Dashboard

This dashboard provides comprehensive visualization and management of your CAD standards mapping system:

  • Name Mappings (DXF ↔ Database): Bidirectional translation between DXF file names and canonical database names for blocks, details, hatches, materials, and keynotes
  • Project Relationships: Semantic connections between elements within projects (keynotes→blocks→details→materials)
  • Multi-Client Support: Scoped mappings for different clients and projects
  • AI-Powered Search: Full-text search across all mappings with weighted relevance
