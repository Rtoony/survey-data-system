About Data Manager

The Data Manager allows you to maintain your CAD standards data through a user-friendly interface. You can:

  • Add/Edit/Delete individual records through the web interface
  • Import CSV files to bulk upload or update data (upsert mode)
  • Export CSV files to backup or share your standards
  • Search and filter records to find what you need quickly
Note on CSV Imports: CSV uploads perform upsert operations (add new or update existing records). They do not delete existing data. If you need to purge a table before uploading, use the Schema page or contact your database administrator.
