About Project Operations

Project Operations helps you manage the lifecycle of your engineering projects:

  • Project Management: Create and organize projects with client information
  • Drawing Management: Track CAD drawings, versions, and metadata
  • Sheet Set Organization: Group drawings into deliverable packages
  • Standards Assignment: Link global standards to specific projects
  • Compliance Tracking: Monitor adherence to assigned standards
Workflow Tip: Start by creating a project, then add drawings and assign relevant standards from your library. Use the Compliance Dashboard to ensure quality.
